# proportion

### tiny, easy jquery proportion helper
----------------------------

Use...

       $("element").proportion(16,9);

...to set the element's proportion to 16 : 9.<br>
Replace "16" and "9" with whatever values you want.
<br>
<br>

Use without values...

       $("element").proportion();

...or with equal values, like...

       $("element").proportion(1,1);

...to get a square.
